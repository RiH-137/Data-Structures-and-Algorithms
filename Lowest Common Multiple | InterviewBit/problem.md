Problem Description
 
 

Find Lowest Common Multiple of given two integers.


Problem Constraints
1 <= A <= 109
1 <= B <= 109


Input Format
First argument is an integer A.
Second argument is an integer B.


Output Format
Return an integer.


Example Input
Input 1:
A = 6
B = 4
Input 2:

A = 1
B = 11


Example Output
Output 1:
12
Output 2:

11
